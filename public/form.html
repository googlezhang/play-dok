Authentication

Some minimal parameters are required to call Applidok merge feature, to get at least PDF unchanged/unfilled. These requirements are Applidok credentials: application token, and template ID.

It can be provided either by application configuration (e.g. conf/application.conf), are as call parameters.

  • Route /merge1.pdf with credentials from configuration.
  • Route /merge1.pdf with template selected by request parameter (applidok_template).
  • Route /merge2.pdf with template specified as function parameter.

Area values

Values to be filled in template area can be given using Play Dok either as function parameters (in your controller) or from field values (form).
